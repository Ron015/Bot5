## 1. update and upgrade!
```
sudo apt update
sudo apt upgrade
```

## 2. install the following to trouble shoot problems:
```
sudo apt default-jdk install apt-transport-https ca-certificates wget dirmngr gnupg software-properties-common
```

3. retry again!
